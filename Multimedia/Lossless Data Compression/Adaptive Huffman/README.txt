HOW THE CODE WORKS:

A- TO COMPRESS:
1- Create a file and write the content that you'd like to compress in this file.
2- When you run the program choose the file that you want to compress and then choose the save location and press 
   the compress button.
3- A new file with the name compressedFile.txt will be created in the save location which contains the coded content
   of the original file.

B- TO DECOMPRESS:
1- You have to choose the compressedFile.txt or the file which has the coded content and then choose the save location
   of the file then press the decompress button.
2- A new file with the name decompressedFile.txt will be created in the save location which contains the original 
   content of the file.
